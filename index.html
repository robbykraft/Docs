Rabbit Ear

grab 

var myOrigami = ear.origami()

This is a Javascript library for modeling origami.

It can fold a crease pattern.

Design based on rules of origami.

Make Diagrams.

Try it

Check out these docs, this button is located next to all the sketches, click on it to reveal the source code.

The code opens in the Rabbit Ear editor, start the editor directly:

code.rabbitear.org

Download

This library is in pre-release.

The library is one file; download it, link the CDN, or grab it from NPM if you are in Node.

rabbit-ear.js (130kb)
https://robbykraft.github.io/Origami/rabbit-ear.js
npm i rabbit-ear

Usage 

<!-- BROWSER -->
<!DOCTYPE html>
<script src="./rabbit-ear.js"></script>
<script>
var origami = ear.origami();
</script>

// NODE JS
const ear = require("rabbit-ear");
var origami = ear.origami();
