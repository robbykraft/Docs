Rabbit Ear

grab

This is a Javascript library for modeling origami.

Three Categories

This library is comprised of three smaller libraries: a math library, a graph library, and a library for drawing SVGs.

Download

Two versions of the library, full (150k), and core (80k) that excludes all SVG-drawing and rendering methods. Both versions work in Node JS.
